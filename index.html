Conexões de rede

Código da UFCD: 0771

Interligação de dispositivos ou sistemas para permitir a comunicação e a transferência de dados entre eles, possibilitando a formação de uma infraestrutura de comunicação eficaz.

Modelo de Rede

Os modelo de rede são formatos de estrutura, física e lógica, de uma rede e a escolha de um modelo depende do que se quer fazer com as máquinas.



Tipos de Modelos de Red

    1. Modelo de Rede de Computadores:

    No contexto de redes de computadores, um modelo de rede é uma representação abstrata da estrutura e funcionamento de um sistema de comunicação. Modelos como o OSI (Open Systems Interconnection) e o TCP/IP são exemplos fundamentais utilizados para compreender e projetar arquiteturas de redes. Estes modelos delineiam a interação entre diferentes componentes, como computadores, roteadores, switches e links de comunicação.

  • 1.1 Modelo OSI:

    O modelo OSI é uma estrutura de sete camadas que descreve as funções necessárias para a comunicação eficiente em uma rede. Cada camada possui responsabilidades específicas, desde a transmissão física dos dados até a aplicação e interação com o usuário final.

  • 1.2 Modelo TCP/IP:

    O modelo TCP/IP, por sua vez, é amplamente adotado na internet e consiste em quatro camadas: Rede, Transporte, Aplicação e Link. Ele define os protocolos de comunicação essenciais para a transferência de dados entre dispositivos conectados à rede.

    2. Modelo de Rede Neural:

    No domínio da inteligência artificial, um modelo de rede muitas vezes se refere a uma rede neural. Inspirada pelo funcionamento do cérebro humano, uma rede neural é um modelo computacional que utiliza camadas de neurônios artificiais conectados. Cada conexão possui um peso ajustável durante o treinamento do modelo.

  • 2.1 Arquitetura de Rede Neural:

    As redes neurais podem ter diversas arquiteturas, como perceptrons simples, redes feedforward, redes recorrentes e redes convolucionais. Essas arquiteturas são aplicadas em tarefas como reconhecimento de padrões, processamento de linguagem natural e visão computacional.


Tipos de rede e de ligação



Tipos de Redes



  • 1. LAN (Rede Local):

    Uma LAN é uma rede que se estende por uma área geográfica relativamente pequena, como um escritório, casa ou campus. Dispositivos conectados a uma LAN podem compartilhar recursos, como impressoras e servidores de arquivos.

  • 2. WAN (Rede de Área Ampla):

    Ao contrário da LAN, uma WAN cobre uma área geográfica mais extensa, como uma cidade, país ou até mesmo globalmente. A internet é um exemplo de uma WAN.

  • 3. MAN (Rede Metropolitana):

    Uma MAN está em algum lugar entre uma LAN e uma WAN, cobrindo uma área metropolitana. Ela é utilizada para interconectar várias LANs dentro de uma cidade.

  • 4. PAN (Rede Pessoal):

    Uma PAN é uma rede pessoal utilizada para conectar dispositivos em curta distância, como entre um smartphone e um computador usando Bluetooth.

  • 5. VPN (Rede Virtual Privada):

    Uma VPN é uma rede segura construída sobre uma infraestrutura de rede pública, como a internet. Ela é usada para conectar remotamente dispositivos ou redes de forma segura.

  • 6. Intranet:

    Uma intranet é uma rede privada baseada em tecnologias da internet, usada dentro de uma organização para facilitar a comunicação e o compartilhamento de recursos.

  • 7. Extranet:

    Similar a uma intranet, uma extranet estende a rede a usuários externos, como parceiros de negócios, permitindo o acesso controlado a determinados recursos.


Modos de Ligação



  • 1. Conexão Ponto a Ponto:

    Nesse tipo de ligação, dois dispositivos estão diretamente conectados, facilitando a comunicação direta entre eles. Redes ponto a ponto são comuns em configurações de redes domésticas.

  • 2. Topologia em Estrela:

    Na topologia em estrela, todos os dispositivos são conectados a um ponto central, como um switch ou hub. Isso facilita a administração, mas a falha desse ponto central pode afetar toda a rede.

  • 3. Topologia em Anel:

    Na topologia em anel, os dispositivos são conectados em um formato circular. A comunicação ocorre em uma única direção, passando por cada dispositivo até atingir o destino.

  • 4. Topologia em Malha:

    Cada dispositivo em uma topologia em malha está conectado a todos os outros. Isso proporciona redundância e confiabilidade, pois a falha de um caminho de comunicação não impede a comunicação.

  • 5. Topologia em Barramento:

    Nessa topologia, todos os dispositivos compartilham um único canal de comunicação. Embora seja simples, a falha desse canal pode comprometer toda a rede.

  • 6. Topologia em Árvore:

    A topologia em árvore é uma combinação de topologias em estrela e barramento, organizando os dispositivos em camadas hierárquicas.
